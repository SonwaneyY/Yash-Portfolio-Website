# Outputs

This directory contains your finalized, polished content ready to use in your portfolio.

## Organization

Mirror your drafts structure for easy reference:

```
outputs/
├── project-titles/
├── about-page/
├── image-captions/
├── navigation/
└── case-studies/
```

## Best Practices

- **Version your finals** - Add date or version number (e.g., `bio-final-2024-11.md`)
- **Include metadata** - Note when it was created, which command was used
- **Track what works** - Note successful content to inform future refinements
- **Update your style guide** - If you discover new preferences, document them
