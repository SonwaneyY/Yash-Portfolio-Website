### Where Keywords Matter Most:

1. **Page Titles (Title Tags)**
   - Most important for SEO
   - Include primary keyword
   - Keep under 60 characters
   - Format: [Name] | [Role] | [Specialization/Location]
   - Example: "Jane Smith | Product Designer | Healthcare UX"

2. **Meta Descriptions**
   - Appears in search results
   - 150-160 characters
   - Include primary and secondary keywords
   - Call to action
   - Example: "Product designer specializing in healthcare and fintech. I create user-centered designs that drive measurable outcomes. View my work."

3. **H1 Headers**
   - One per page
   - Should include primary keyword
   - Example: "Product Designer Specializing in Healthcare UX"

4. **H2/H3 Headers**
   - Subheadings throughout content
   - Include secondary keywords naturally
   - Example: "User Research & Design Strategy"

5. **Body Content**
   - First 100 words are valuable
   - Natural keyword integration throughout
   - Use variations and related terms
   - Semantic keywords (related concepts)

6. **Image Alt Text**
   - Describe image + include keywords when natural
   - Example: "Product design mockup for healthcare dashboard showing patient data visualization"

7. **URLs**
   - Clean, descriptive URLs
   - Include keywords when possible
   - Example: `/work/healthcare-dashboard-design` not `/project-123`

## SEO Best Practices

### Do:
- Write for humans first, optimize second
- Use keywords naturally in context
- Include variations (product designer, product design, designing products)
- Use semantic keywords (related terms search engines recognize)
- Keep content fresh and updated
- Use descriptive, keyword-rich image file names

### Don't:
- Keyword stuff (repeating keywords unnaturally)
- Sacrifice readability for SEO
- Use keywords that don't match your actual work
- Hide keywords in white text or other black-hat techniques
- Ignore mobile optimization
- Forget about page speed and technical SEO

## Keyword Density Guidelines:

- **Primary keyword**: 1-2% of content (natural mentions)
- **Secondary keywords**: Sprinkled throughout naturally
- **Related terms**: Frequently (semantic SEO)
- **Over-optimization**: If it feels forced, it is

## Content-Specific SEO Tips

### Homepage:
- H1 with primary keyword (your role + specialization)
- Meta description with primary and secondary keywords
- First paragraph should include main keywords
- Clear navigation with keyword-rich anchor text

### About Page:
- Title: [Name] | [Role] Bio
- Include keywords in bio naturally
- Mention industries, skills, methodologies
- Location if relevant for local search

### Project Pages:
- Title: [Project Name] | [Industry] [Design Type]
- Example: "Healthcare Dashboard Redesign | Product Design Case Study"
- H1 with project name + what it is
- H2s for sections (Research, Design Process, Outcomes)
- Keywords in introduction and throughout

### Contact Page:
- Title: Contact [Name] | [Role]
- Location and availability keywords
- Services offered
